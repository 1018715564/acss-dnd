# acss-dnd (atomic css drag and drop)

`acss-dnd`是一款基于原子类 css 特性，实现拖拽布局的一个工具。

## ✨ 特点

- 简单易用
- 导出符合 html 规范代码，利于二次开发
- 响应式
- 多种主题模式

## 💻 在线体验

- [Github](https://meetqy.github.io/acss-dnd/)
- [Gitee](https://meetqy.gitee.io/acss-dnd)

## 项目结构

```
- template   # 存放模板配置
- views      # 模块
    - template              # 模板
    - main                  # 编辑器外部的样式
    - editor                # 画布
    - menu-bar              # 菜单
    - side-bar              # 属性操作
        - file-a            # 一个文件夹相当于一个分类
        - file-b
        ...
```

## 基础库
