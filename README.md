# ultraseven

只需要简单的拖拽，你就能创建一个漂亮的响应式网页。

## ✨ 特点

- 树形结构的 dom 结构，更加符合开发者的布局方式
- 因为基于 tailwind，所以提高了 ui 的下限
- 响应式，拖拽出来的网页天生就是响应式
- 多种选择，无论你是移动端优先，还是 pc 端优先
- 适配暗黑模式
- 不受框架限制，只要支持 html，均可使用
